Interaction Design is about how users engage with your interface:

  • What happens when they click, swipe, drag, or hover?
  • How does the system respond?

üìê Designing States for Interactions

Every interactive component should have:

  1. Default State (e.g., a grey button)
  2. Hover State (e.g., blue background)
  3. Active/Clicked State (e.g., darker blue)
  4. Disabled State (e.g., light grey)
  5. Success/Error State (e.g., green/red)

Create variants in Figma for each state and link with Smart Animate.
