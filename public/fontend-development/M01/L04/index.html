Lesson 04: HTML Best Practices

Module 01: HTML5 Structure and Semantics

Objective

Write clean, maintainable HTML and validate your code to ensure quality and compatibility.

Content

1. Writing Clean HTML

Learn to structure HTML with proper indentation, comments, and consistent naming.

Watch Video (5 min)

2. Validation Tools

Use W3C Markup Validator to check for errors and ensure cross-browser compatibility.

Watch Video (10 min)

3. Hands-On Exercise

Refactor a poorly written HTML page and validate it. Example (before):

<H1>welcome</h1><P>content<BR><img src="img.jpg">

After:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  <h1>Welcome</h1>
  <p>Content</p>
  <img src="img.jpg" alt="Description">
</body>
</html>
