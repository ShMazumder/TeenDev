Lesson 01: JavaScript Basics

Module 04: JavaScript (ES6+) Interactivity

Objective

Learn JavaScript fundamentals like variables, functions, and conditionals to add logic to your websites.

Content

1. Introduction to JavaScript

Understand JavaScript's role in web development and its basic syntax.

Watch Video (5 min)

2. Variables and Control Flow

Use var, let, const, conditionals (if/else), and loops (for, while).

Watch Video (10 min)

3. Hands-On Exercise

Write a script to calculate the sum of numbers. Example:

let sum = 0;
for (let i = 1; i <= 5; i++) {
  sum += i;
}
console.log(sum); // 15

4. Quiz

Test your knowledge with a 5-question quiz on JavaScript basics.
