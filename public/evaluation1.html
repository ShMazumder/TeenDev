Software Engineer Hiring Evaluation

Complete your assigned task and submit your GitHub repository link before the deadline.

1. Web Development Task

  • Create a Student Management System (Add/View/Edit/Delete students).
  • Frontend: HTML, CSS (Bootstrap/Tailwind), JavaScript.
  • Backend: Node.js + Express.js or PHP.
  • Database: MySQL or MongoDB.
  • Bonus: Deploy the app (Netlify/Vercel/etc.).

2. Mobile App Development Task

  • Create a Weather App using Flutter.
  • Use OpenWeatherMap API to fetch current weather.
  • Must have a Search Bar and Weather Result display.
  • Bonus: Create a microservice API backend.

3. UI/UX Design Task

  • Design a Food Delivery App in Figma or Adobe XD.
  • Include: Home, Restaurant List, Dish Details, Cart, Order Summary pages.
  • Create a clickable interactive prototype.

Submission Instructions

  • Upload your work to a public GitHub repository.
  • Include a proper README.md file.
  • Commit your work in stages (not all at once).
  • Submit the GitHub repo link before the deadline.

Evaluation Criteria

  • Technical Skills (25 points)
  • Task Completion (20 points)
  • Code Quality (15 points)
  • GitHub Usage (10 points)
  • Creativity & Bonus Effort (10 points)
  • Problem Solving/Handling Errors (10 points)
  • Presentation (10 points)

Scoring Guide: 85-100 = Ready for Hiring | 70-84 = Almost Ready | 50-69 = Needs Training | Below 50 = Strong Training Recommended
