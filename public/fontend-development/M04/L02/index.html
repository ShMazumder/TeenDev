Objective

Manipulate the DOM using querySelector and event listeners to create interactive elements.

Content

1. Understanding the DOM

Learn how the Document Object Model represents HTML and how to access it.

Watch Video (5 min)

2. Selecting and Modifying Elements

Use querySelector and event listeners to change content and styles dynamically.

Watch Video (10 min)

3. Hands-On Exercise

Create a button that changes text on click. Example:

const button = document.querySelector('#myButton');
button.addEventListener('click', () => {
  document.querySelector('#myText').textContent = 'Clicked!';
});
