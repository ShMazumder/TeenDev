Lesson 03: Forms and Accessibility

Module 01: HTML5 Structure and Semantics

Objective

Create interactive forms and ensure accessibility with proper HTML attributes and ARIA roles.

Content

1. HTML Forms Basics

Learn to create forms using <form>, <input>, <select>, and <textarea> elements.

Watch Video (5 min)

2. Accessibility in Forms

Use labels, ARIA roles, and alt text to make forms accessible to all users.

Watch Video (10 min)

3. Hands-On Exercise

Build an accessible contact form. Example:

<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required aria-required="true">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required aria-required="true">
  <label for="message">Message:</label>
  <textarea id="message" name="message" aria-describedby="message-desc"></textarea>
  <p id="message-desc">Enter your message here.</p>
  <button type="submit">Submit</button>
</form>
