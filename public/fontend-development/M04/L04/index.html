Lesson 04: Fetching APIs

Module 04: JavaScript (ES6+) Interactivity

Objective

Fetch and display data from public APIs using the Fetch API and handle JSON responses.

Content

1. Introduction to APIs

Understand what APIs are and how to use the Fetch API for HTTP requests.

Watch Video (5 min)

2. Handling JSON Data

Parse JSON responses and display data in the DOM.

Watch Video (10 min)

3. Hands-On Exercise

Fetch and display data from a public API. Example:

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    document.querySelector('#output').textContent = data[0].name;
  });
