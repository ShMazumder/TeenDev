Content

2. Typography and Gradients

Apply Google Fonts and CSS3 gradients to enhance visual design.

Watch Video (10 min)

3. Hands-On Exercise

Style a navigation bar with hover effects. Example:

nav {
  background: linear-gradient(to right, #1e3a8a, #3b82f6);
}
nav a {
  color: white;
  padding: 10px;
}
nav a:hover {
  background: rgba(255, 255, 255, 0.2);
}
