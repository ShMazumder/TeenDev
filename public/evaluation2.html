Assignment Portal

Deadline:

1. Web Development Task

  • Build a Student Management System (Add/View/Edit/Delete Students).
  • Frontend: HTML, CSS (Bootstrap/Tailwind), JavaScript.
  • Backend: Node.js + Express.js or PHP.
  • Database: MySQL or MongoDB.
  • Bonus: Deploy the app on Netlify/Vercel.

2. Mobile App Development Task

  • Develop a Weather App using Flutter.
  • Fetch current weather using OpenWeatherMap API.
  • Features: Search by City, Display Weather Details.
  • Bonus: Create a microservice API backend.

3. UI/UX Design Task

  • Design a Food Delivery App in Figma or Adobe XD.
  • Include Home, Restaurant List, Dish Details, Cart, Order Summary pages.
  • Create a clickable interactive prototype.

Submission Instructions

  • Upload your project to a Public GitHub Repository.
  • Ensure your project has a proper README.md.
  • Commit your work in stages (not all at once).
  • Submit your GitHub repository link using the button below.
