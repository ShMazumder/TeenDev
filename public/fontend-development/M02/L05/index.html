Apply media queries and relative units (rem, vw) to create layouts that adapt to different screen sizes.

2. Media Queries and Units

Use media queries and relative units like rem and vw for responsive layouts.

Watch Video (10 min)

3. Hands-On Exercise

Create a responsive layout. Example:

.container {
  font-size: 1rem;
}
@media (max-width: 600px) {
  .container {
    font-size: 0.8rem;
  }
}
