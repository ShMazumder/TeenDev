Objective

Build engaging animations using @keyframes for hero sections, loaders, and microinteractions.

Content

1. Introduction to Keyframes

Learn how to define animations using @keyframes and the animation property.

Watch Video (5 min)

2. Creating Animations

Apply keyframes to create effects like fade-ins, spins, and bounces.

Watch Video (10 min)

3. Hands-On Exercise

Create a loading spinner with keyframes. Example:

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
.spinner {
  animation: spin 1s linear infinite;
  border: 4px solid #f3f3f3;
  border-top: 4px solid #2563eb;
  border-radius: 50%;
  width: 40px;
  height: 40px;
}
